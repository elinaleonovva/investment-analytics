## Автор
Леонова Элина, ИКБО-10-23

# Описание
Market Vision — веб-приложение для отслеживания рыночных данных и ведения инвестиционного портфеля.

Проект позволяет:
- регистрировать и авторизовывать пользователей;
- просматривать данные по акциям, валютам и драгоценным металлам;
- обновлять рыночные данные;
- создавать инвестиционные портфели;
- добавлять сделки покупки и продажи;
- анализировать текущую стоимость портфеля, вложенные средства, PnL и доходность;
- формировать PDF-отчёт по портфелю.

Backend проекта разделён на несколько Django-приложений:
- `authentication` — регистрация, вход, JWT-аутентификация;
- `fixings` — рыночные данные, котировки акций, валют и металлов;
- `portfolio` — портфели, сделки, аналитика и отчёты.

# Запуск проекта

1. Клонируйте репозиторий:

```powershell
git clone https://github.com/elinaleonovva/investment-analytics
cd investment-analytics
```

2. Создайте файл `.env` на основе шаблона:

```powershell
Copy-Item .env.example .env
```

3. При необходимости проверьте и отредактируйте параметры в `.env`.

4. Запустите контейнеры:

```powershell
docker compose up --build
```

5. После запуска приложение будет доступно по адресам:

- frontend: [http://localhost](http://localhost)
- backend API: [http://localhost:8000](http://localhost:8000)

Backend-контейнер при старте автоматически:
- применяет миграции;
- создаёт валюты;
- создаёт список акций;
- загружает рыночные данные;
- запускает приложение через Gunicorn.
